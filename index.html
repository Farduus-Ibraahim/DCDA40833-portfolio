04 — Semester Reflection

Looking Back & Forward

I

What I Learned

How to set up a GitHub repository, edit HTML and CSS files, and publish a website using GitHub Pages. This was my first time connecting all of these tools together, and it helped me understand how a portfolio site actually works behind the scenes.

II

Challenges I Overcame

Navigating GitHub and knowing where certain settings were located, especially when enabling GitHub Pages. I worked through this by carefully following the instructions, revisiting steps when I got stuck, and learning to be more patient with the process.

III

How I Grew

My understanding of digital culture and data analytics became more practical and hands-on. I started to see how technical skills like HTML, CSS, and GitHub are not just tools, but part of how digital spaces are created, shared, and experienced.

IV

Looking Ahead

I plan to continue building and refining my portfolio with class projects, data visualizations, and reflections — using this site as a professional space as I prepare for internships, graduate school, and future career opportunities.
