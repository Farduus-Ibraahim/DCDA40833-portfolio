Hometown Map

Interactive Hargeisa Map

For this lab, I created an interactive hometown map centered on Hargeisa, Somaliland. I designed a custom basemap in Mapbox Studio, built a dataset of meaningful places from my hometown, and then used Python and Folium to turn that data into an interactive web map. Each marker represents a place that feels important to me, and together they show a more personal view of Hargeisa.

One part I struggled with was geocoding. Some locations were being placed in the wrong places, including Ethiopia and even Australia, because Somaliland addresses are not always standardized in global mapping systems. To fix that, I switched to using latitude and longitude coordinates directly in my dataset, which made the map much more accurate.
